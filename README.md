# Flip the Switch: Feature Flags and the Power of Open Standards

A talk by Simon Schrottner — OpenFeature maintainer, CNCF Ambassador.

Feature flags sound simple until your codebase is welded to a proprietary SDK and
the vendor raises prices, gets acquired, or stops caring. This talk uses
OpenFeature as a case study in how an **open standard** changes not just the
technology but how a community collaborates — and the room that creates for new
ideas, tools, and startups.

Sibling talk (the technical deep-dive): [Fun with Flags](https://github.com/aepfli/openFeatureTalk).
